# Swipe VS

無限スワイプ型の2択投票プラットフォーム（フロントエンド・プロトタイプ）

Tinder風の左右スワイプ投票と、TikTok風の縦スクロール、掲示板風コメントを組み合わせた Web アプリです。

## 主な機能

- 左右スワイプで A / B に投票
- 投票後の結果グラフ表示
- コメント・いいね・お題作成
- 匿名 ID によるマイページ（投票履歴・作成お題）
- お題ごとのシェア URL（`?topic=お題ID`）
- 未読お題の優先表示
- カテゴリ付きモックお題（仕事 / グルメ / ライフスタイル など）

## 技術スタック

- React 19 + TypeScript
- Vite 8
- framer-motion
- Local Storage（永続化）

## セットアップ

```bash
npm install
npm run dev
```

### スマホ実機で確認（同一 Wi-Fi）

```bash
npm run dev:host
```

表示された `Network` の URL（例: `http://192.168.x.x:5173/`）をスマホのブラウザで開きます。

## データの保存について

ブラウザの Local Storage に以下を保存します。

- 匿名ユーザー ID
- 投票・いいね・コメント・作成お題
- フィード位置・既読お題

**注意:** ブラウザのデータを消すと履歴も消えます。複数端末間の同期は未対応です。

## シェア URL

お題ごとに次の形式の URL を共有できます。

```
https://example.com/?topic=1
```
